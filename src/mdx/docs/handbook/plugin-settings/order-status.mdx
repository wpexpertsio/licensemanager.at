The order status tab contains settings related to triggering the license key delivery.

![The “License key delivery” settings section](/images/docs/plugin-settings/order-status/license-key-delivery.png)
<p className={"image-desc"}>The “License key delivery” settings section</p>

The “**Define license key delivery**” setting allows the user to configure which order status changes will trigger the
license key delivery. By default, only “Completed” is enabled.
