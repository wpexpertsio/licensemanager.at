description:
lastModified: 14.03.2021

Should you ever need to resend the license keys to the customer (if you had to
edit or replace the license key, for example), you can do so by selecting the
“**Send license key(s) to customer**” order actions on the right-hand side of
the screen, as shown below.

![License keys sold by a WooCommerce order are displayed inside the order itself](../../../../images/docs/woocommerce-orders/resend-license-keys.png)
