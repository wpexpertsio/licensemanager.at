```php
function lmfwc_get_license_meta(int $licenseId, string $metaKey, bool $single = false)
```

## Description

Retrieves single or multiple license meta entries.

## Parameters

**$licenseId** *(string) (Required)*<br/>
The ID of the license key.

**$metaKey** *(string) (Required)*<br/>
Name of the meta entry.

**$single** *(bool) (Optional)*<br/>
If multiple meta entries exists, setting this parameter to **true** will retrieve
all of them.

## Return value

*(mixed|mixed[]|bool)*<br/>
Meta value on success, false on failure.
