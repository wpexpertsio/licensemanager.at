```php
function lmfwc_add_license_meta(int $licenseId, string $metaKey, mixed $metaValue)
```

## Description

Adds a new meta entry to a license, if it already exists a new row in the
database table will be added.

## Parameters

**$licenseId** *(string) (Required)*<br/>
The ID of the license key.

**$metaKey** *(string) (Required)*<br/>
Name of the meta entry.

**$metaValue** *(mixed) (Required)*<br/>
Value of the meta entry.

## Return value

*(mixed|bool)*<br/>
Meta value on success, false on failure.
