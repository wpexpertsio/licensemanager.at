```
function lmfwc_get_licenses(array $query)
```

## Description

Retrieves multiple existing licenses from the database.

## Parameters

**$query** *(array) (Required)*<br/>
Key/value pairs. The keys will be used to  match the license table columns
names, and the values will be used to match the values of those columns.


## Return value

*(bool|LicenseManagerForWooCommerce\Models\Resources\License[])*<br/>
License model array on success, false on failure.
