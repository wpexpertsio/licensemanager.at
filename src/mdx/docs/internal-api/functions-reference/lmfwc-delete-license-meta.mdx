```
function lmfwc_delete_license_meta(int $licenseId, string $metaKey, mixed $metaValue = null)
```

## Description

Deletes a license meta entry. If the **$metaValue** parameter is passed, the
function will look for the meta entry with a matching value to delete.

## Parameters

**$licenseId** *(string) (Required)*<br/>
The ID of the license key.

**$metaKey** *(string) (Required)*<br/>
Name of the meta entry.

**$metaValue** *(mixed) (Optional)*<br/>
Value of the meta entry.

## Return value

*(bool)*<br/>
True on success, false on failure.
