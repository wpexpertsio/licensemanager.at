## Description

Retrieves a single generator by its ID. The response contains the generator
data object.

## Request URL

<div className={"doc-request-example"} style={{ marginBottom: '1em' }}>
  <div className={"method get"}>GET</div>
  <div className={"url"}>/wp-json/lmfwc/v2/generators/<span className={"variable"}>&#123;generator_id&#125;</span></div>
</div>
