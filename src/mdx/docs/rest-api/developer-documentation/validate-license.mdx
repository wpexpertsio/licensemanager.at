## Description

Checks the current activation status of a license key. The response will
contain the number of activations, the maximum activation count, and the
remaining activations.

## Request URL

<div className={"doc-request-example"} style={{ marginBottom: '1em' }}>
  <div className={"method get"}>GET</div>
  <div className={"url"}>/wp-json/lmfwc/v2/licenses/validate/<span className={"variable"}>&#123;LICENSE&#125;</span></div>
</div>

## Response example

```
{
  "success": true,
  "data": {
    "timesActivated": 0,
    "timesActivatedMax": 999,
    "remainingActivations": 999
  }
}
```
