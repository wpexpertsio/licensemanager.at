## Description

Performs an update of the license key. The request will not update key values
that aren't present in the request body, however if they are present their
value will be updated, even if itâ€™s a null value. The request will return the
updated license key data object.

## Request headers

```
Content-Type: application/json
```

```
{
  "order_id": null,
  "product_id": null,
  "license_key": "THE-PRETENDER",
  "valid_for": null,
  "times_activated_max": 999,

}
```
