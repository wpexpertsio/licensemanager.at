## Description

Increments the `times_activated` value by one (1). The plugin will check if
there is a `times_activated_max` value, if there is and if the
`times_activated` value has not reached the limit set by `times_activated_max`,
then the `times_activated` will be incremented by 1, and the updated license
key data object will be returned.

## Request URL

<div className={"doc-request-example"} style={{ marginBottom: '1em' }}>
  <div className={"method get"}>GET</div>
  <div className={"url"}>/wp-json/lmfwc/v2/licenses/activate/<span className={"variable"}>&#123;LICENSE&#125;</span></div>
</div>
