## Description

Retrieves all currently available generators. The response will contain an
array of generator data objects.

## Request URL

<div className={"doc-request-example"} style={{ marginBottom: '1em' }}>
  <div className={"method get"}>GET</div>
  <div className={"url"}>/wp-json/lmfwc/v2/generators</div>
</div>
